# helm-chart prompt 文档

当涉及项目 helm-chart values、template 编写，或是部署更新时，请 Agent 参考本文档中的信息，你可以将关键的指令与提示总结并输出到对话 context 中，或者提示自己在设计方案和实现时再次查阅文档，方便你在全程记住它们并依照它们行动。

- 由于配置中存在各种密钥，一定不能将相关内容上传至公网。

# 更新部署

`act-gpu-cluster` 仓库中包含了生产集群和小集群的 `values.yaml` ，用于自动更新其中镜像标签的脚本，以及提供了更新命令的 README。

- 如果不涉及 helm-chart 版本的更新，直接执行自动更新脚本，再更新至生产集群即可。
- 如果涉及 helm-chart 版本的更新，则必须要反复提醒用户修改 README 中对应的版本，然后推送修改，避免出现版本冲突。
